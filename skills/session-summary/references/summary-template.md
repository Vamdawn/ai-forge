# 会话摘要模板

```markdown
# 会话摘要 — YYYY-MM-DD HH:MM

## 基本信息
| 字段 | 内容 |
|------|------|
| 日期 | YYYY-MM-DD |
| 工作目录 | /path/to/project |
| 交互轮次 | N 轮（用户消息 N 条 / 助手回复 N 条） |
| 预估 Token 消耗 | ~N,000 tokens |
| 整体耗时 | 约 N 分钟 |
| 主要语言 | 中文 / 英文 |

## 会话概要

### 核心议题
> 一句话概括本次会话的主要目标

1. **议题一** — 简短描述
2. **议题二** — 简短描述

### 决策与结论
- ✅ **已决定**：[决策内容]
- ✅ **已决定**：[决策内容]

### 文件变更
| 文件路径 | 操作 | 说明 |
|---------|------|------|
| `path/to/file` | 创建/修改/删除 | 变更说明 |

### 代码与命令摘要
- 执行了什么关键命令
- 实现了什么功能

### 重要备注
> 任何需要下次会话记住的上下文、注意事项或风险

## 会话明细

**[1]** 用户原始输入内容
↳ 助手操作与回复摘要

**[2]** 用户原始输入内容
↳ 助手操作与回复摘要

---
*由 session-summary skill 自动生成*
```
